Typescript Functions

Basic JS Function Declaration

A function is a set of statements to perform a specific task. Functions organize the program into logical blocks of code.


function  function_name( ) {{ '{' }} } // function definition
function_name() // function invocation

Positional parameters in Typescript Function

  • Parameters can have a specified type by using the syntax:
    • param_name: type
  • Parameters without a specified type are considered of type any
  • Parameters can be made optional with '?' by using the syntax
    • param_name?: type
  • Parameters can be given a default value with '=' just like in regular JS functions.
  • Note that a parameter can't be optional and have a default value

function  function_name(p1: string = "hello", p2?: number) {{ '{' }}   
    console.log(p1);                                
    console.log(p2);                                
}

Typescript Function Types

  • Function types can be written as a call signature in an object type:
    • The following code is defining a function type with one parameter, named a (of type string) that doesnâ€™t return anything:
    • type GreetFunction = {{ '{' }}(a: string): void;};
  • The below code is exactly the same code as above but written in shorthand form (similar to an arrow function):
    • (a: string) => void
  • A type alias can be used to name a shorthand function type:
    • type GreetFunction = (a: string) => void; 

// define function type  
type GreetFunction = (a: string) => void;     

// implement function of type GreetFuncti
function printToConsole(s: string) {{ '{' }}    
    console.log(s);                     
}                                           

// implement function that gets a parameter of type GreetFunction
function greeter(fn: GreetFunction) {{ '{' }}   
    fn("Hello, World");                 
}                                           

// invoke function greeter               
greeter(printToConsole);
